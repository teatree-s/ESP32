# 2. 使い方

ESP32 を使ったプログラムを置いています。各プロジェクトにて詳細確認ください。

- ESP-WROOM-32D
  - BLE_Wiimote : Wii リモコンで画面に表示している画像を変更できるプロジェクト
  - M5Stack_Avatar : オープン仕様スタックチャンの M5Stack_Avatar を使ったプロジェクト
- NodeMCU-ESP32S
  - BLE_DrawImage : BLE リモコンで画面に表示している画像を変更できるプロジェクト
